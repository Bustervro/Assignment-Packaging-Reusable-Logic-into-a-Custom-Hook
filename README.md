# useWindowSize React Lab

This project demonstrates a custom React hook called useWindowSize.

## Features

- Tracks browser window width
- Tracks browser window height
- Changes layout based on screen size
- Shows mobile layout under 768px
- Shows desktop layout at 768px and above
- Cleans up the resize event listener

## Installation

npm install

## Run Project

npm run dev
